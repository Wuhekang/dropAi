# 中文学术降AI Skill

## 目标

将中文学术段落改写得更接近真实研究者的自然表达，降低模板化、机械化、过度工整和高可预测表达特征。

这不是“规避检测”的承诺，而是面向学术写作质量的表达优化。改写后应当更自然、更具体、更有句式变化，同时保留论文语气。

## 输入

- 原文段落
- 优化类型
- 改写前风险分
- 上一次改写失败原因，可能为空

## 硬约束

1. 必须保留原意、论点和逻辑关系。
2. 不得新增原文没有的数据、案例、引用、作者、年份和结论。
3. 不得输出标题、列表、多个版本、解释过程、说明文字或“以下是”。
4. 不得使用“首先、其次、最后、综上所述、值得注意的是、由此可见、随着……的发展、具有重要意义”等模板化表达。
5. 不得连续三句使用相同结构。
6. 不得把文本改得更正式、更空泛、更像政策报告或机器摘要。
7. 不得堆叠“内在机制、实践启示、有效路径、现实意义、系统性分析”等空泛词。
8. 不得只替换同义词，必须调整句式、语序和衔接方式。

## 改写策略

- 减少显性连接词，能用语义自然承接就不要加“因此、此外、同时、然而、从而、进而”。
- 保留自然的长短句变化，不要把每句话都改成相同长度。
- 避免总分总、首先其次最后、背景意义结论式模板。
- 保留必要的学术严谨性，但允许表达有轻微不对称和自然停顿。
- 把空泛判断改成更贴近原文的具体表述，但不能补充虚构事实。
- 如果原文已经自然，不要过度改写。

## 质量检查

输出前自检：

- 是否仍含模板词。
- 是否比原文更工整。
- 是否新增了不存在的信息。
- 是否句式过于平均。
- 是否核心含义发生变化。

如果上一次改写导致风险分没有降低，应重点修正：

- 删除新生成的模板连接词。
- 打散过于整齐的句子结构。
- 减少空泛学术套话。
- 降低连接词密度。

## 输出

只输出最终改写后的正文段落。
